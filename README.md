# Data Science & Artificial Intelligence Projects

This repository contains personal projects developed to strengthen my skills in data analysis, visualisation, and applied machine learning using Python.

The objective of these projects is to build a solid practical foundation in data processing, statistical modelling, and interactive visualisation as preparation for advanced study in Artificial Intelligence.

---

## Projects

### 1. Olympic Games Interactive Dashboard
Interactive dashboard built using Python, Pandas, Plotly and Dash.

- Exploratory Data Analysis on ~270,000 records
- Data cleaning and preprocessing
- Interactive filtering and aggregation
- Linear regression visualisation
- Geographic mapping (choropleth)
- Callback-based interactivity with Dash

File: `Dashboard_Olimpico_Interactivo.ipynb`
